# Description
`ezneis` is a library that help to use easily the [NEIS Open API](https://open.neis.go.kr/).  
supports both sync and async, all data has been parsed, so it can be used easily.


# Install & Update
```shell
# Install
pip install ezneis

# Update
pip install --upgrade ezneis
```
