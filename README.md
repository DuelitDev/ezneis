# ezneis


# 설명
`ezneis`는 [나이스 Open API](https://open.neis.go.kr/)를 쉽게 사용할 수 있도록 도와주는 라이브러리입니다.  
동기, 비동기 모두 지원하며 모든 데이터가 파싱되어 간단하게 데이터를 이용할 수 있습니다.


# 설치 및 업데이트
```shell
# 설치
pip install ezneis

# 업데이트
pip install --upgrade ezneis
```
